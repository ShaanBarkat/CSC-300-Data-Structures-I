CSC 300/Data Structures in Java I
Assignment: Counting letters in a text file

20 points

Setting up

First, please read the section "Assignments and grading" in the syllabus. Then, following the steps for creating programs, create a package called countLetters.

Specifications

Write a program called CountLetters.java that prompts the user for the name of a text file, reads in all of the text in that file, and then computes and prints a table of letter frequencies. For example, if the file is the text of the U.S. constitution, which you downloaded for the CountLetters program, the program will print:

Submission instructions

When done correctly, you will have a single program CountLetters.java in a package called countLetters. This is located in your file system under the workspace folder you created for Eclipse. Starting at that folder, navigate to algs4/src/countLetters. Zip that folder (that is, zip the folder countLetters NOT just the file CountLetters.java) into a zip file called countLetters.zip and submit that zip file into the D2L dropbox for this assignment no later than its due date.

Four very important points to remember:

  1. Your name must appear in a comment at the top of the program. You can insert it either in this style: 
    // Author: Alan Turing
    or this style: 
    /*
 * Author: Alan Turing
 */
  2. Give the files the names EXACTLY as specified, including the capitalization. Not doing so will cause points to be deducted.
  3. A submission may be graded as soon as it appears in its drop box. In other words, don't submit until you're sure that you want the program graded.
  4. Just after the due date, zero points will automatically be given any assignment not submitted.
