CSC 300/Data Structures in Java I
Creating programs

Each program presented in class or that you will write for an assignment will contain one or more Java files. The files that are part of a program will be contained in a Java package with a name given as part of the assignment. So to write a program you must first create a package in Eclipse.

  1. Start up Eclipse. In the Package Explorer pane you should see the algs4 package. If not, please see the instructions in the Eclipse document.
  2. If not already expanded, click on the arrow next to the algs4 folder.
  3. Right click on the src folder and select New > Package.
  4. In the dialog box that appears type in the name of the package, which should begin with a lowercase letter. For assignments, I will specify the name you should use.

For each file that will be part of a package, do the following:

  1. Right click on the package icon and select New > Class.
  2. In the dialog box that appears type in the class name required by the assignment. This will always be a name with an uppercase first letter.
  3. In the same dialog box, locate the heading 
    Which method stubs would you like to create?
    Underneath it click the checkbox next to 
    public static void main(String[] args)
  4. In the editing window you will see, for example, code like this appear: 
    package Hello;

public class Hello {

    public static void main(String[] args) {
        // TODO Auto-generated method stub

    }

}
  5. You are now ready to edit the code to do what you need. Now would be a good time to add the required comment at the top containing the name of the program's author. For example: 
    package Hello;
/*
 * Author: Alan Turing
 */

public class Hello {

    public static void main(String[] args) {
        // TODO Auto-generated method stub

    }

}
