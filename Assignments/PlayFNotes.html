CSC 300/Data Structures in Java I
Assignment: Play three F notes

10 points

Setting up

First, please read the section "Assignments and grading" in the syllabus. Then, following the steps for creating programs, create a package called playFNotes.

Specifications

Using the program classes NBCTones and SITTones in the week 1 examples package, write a program class called PlayFNotes that plays in order the notes F4, F5, and F6, each for one and a half seconds. The frequency of F4 is 349.228Hz. Look up the other two. You will need the playTone method so simply copy it from one of the example programs into your program. Note that the second parameter, duration, should be the number of seconds to play the tone.

Submission instructions

When done correctly, you will have a single program PlayFNotes.java in a package called playFNotes. This is located in your file system under the workspace folder you created for Eclipse. Starting at that folder, navigate to algs4/src/playFNotes. Zip that folder (that is, zip the folder playFNotes NOT just the file PlayFNotes.java) into a zip file called playFNotes.zip and submit that zip file into the D2L dropbox for this assignment no later than its due date.

Four very important points to remember:

  1. Your name must appear in a comment at the top of the program. You can insert it either in this style: 
    // Author: Alan Turing
    or this style: 
    /*
 * Author: Alan Turing
 */
  2. Give the files the names EXACTLY as specified, including the capitalization. Not doing so will cause points to be deducted.
  3. A submission may be graded as soon as it appears in its drop box. In other words, don't submit until you're sure that you want the program graded.
  4. Just after the due date, zero points will automatically be given any assignment not submitted.
