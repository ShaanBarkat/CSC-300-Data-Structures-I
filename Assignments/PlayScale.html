CSC 300/Data Structures in Java I
Assignment: Play notes from file

10 points

Setting up

First, please read the section "Assignments and grading" in the syllabus. Then, following the steps for creating programs, create a package called playScale.

Specifications

You are given a file scalenotes.txt that contains, in order, the floating point frequencies of the notes on a piano. Place this file into the data folder on Eclipse.

Using the program class FileAverage for an example of how to read a sequence of values from a file, write a program class called PlayScale that reads those frequencies into an array and then plays each of them for 1/8 of a second. Recall that floating point values are held by a variable of type double in Java.

Submission instructions

When done correctly, you will have a single program PlayScale.java in a package called playScale. This is located in your file system under the workspace folder you created for Eclipse. Starting at that folder, navigate to algs4/src/playScale. Zip that folder (that is, zip the folder playScale NOT just the file PlayScale.java) into a zip file called playScale.zip and submit that zip file into the D2L dropbox for this assignment no later than its due date.

Four very important points to remember:

  1. Your name must appear in a comment at the top of the program. You can insert it either in this style: 
    // Author: Alan Turing
    or this style: 
    /*
 * Author: Alan Turing
 */
  2. Give the files the names EXACTLY as specified, including the capitalization. Not doing so will cause points to be deducted.
  3. A submission may be graded as soon as it appears in its drop box. In other words, don't submit until you're sure that you want the program graded.
  4. Just after the due date, zero points will automatically be given any assignment not submitted.
