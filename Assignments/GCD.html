CSC 300/Data Structures in Java I
Assignment: Compute the GCD of two integers

10 points

Setting up

First, please read the section "Assignments and grading" in the syllabus. Then, following the steps for creating programs, create a package called gcd.

Specifications

Using the LotteryOdds class in the week 1 examples package as an example, write a class called GCD that prompts the user for two integers and then computes and prints their greatest common divisor (GCD).

Here is a Python version of a program to do that: 

def gcd(a, b):
    if b == 0:
        return a
    return gcd(b, a%b)

a = int(input("Enter a: "))
b = int(input("Enter b: "))
print("The gcd is", gcd(a,b))
Note that the main part of the Python program is the last three lines and that, in Java, the main code goes into the main method, as in LotteryOdds.
