Specifications

In class we discussed the Date class. There is a Date class available in the algs12 package. Using it (which you can access with the statement import algs12.Date) write a program called PrintCalendar that prompts the user for a start date and an end date and then prints all of the dates between them (inclusive), with seven tab-separated dates on each line. Here is an example of a run of the program:

To help with this, here is (part of) the API for the algs12.Date class:

  • public Date(int month, int day, int year): This constructs a Date object from the values given.
  • public Date(String dateString): This constructs a Date object from a String formatted as an American date, e.g., "5/6/2017" would create a Date object for May 6th, 2017.
  • public Date next(): When called with a Date object, this returns a new Date object representing the next day.
  • public boolean equals(Date that): Returns true if the Date object for which it is called is equal to the that Date object.
  • public boolean isBefore(Date that): Returns true if the Date oject for which it is called is strictly before the that Date object.

Before printing the calendar, your program should check that the first date is not after the second date. If it is, it should print an error message and not print the calendar.
